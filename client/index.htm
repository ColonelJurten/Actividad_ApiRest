Clima — Consumo de API REST con secretos

El cliente llama a un proxy Express; la API key vive en el servidor.

¿Cómo se protege el secreto?

  • La API key está en server/.env, ignorado por Git.
  • El navegador solo llama a /api/weather del servidor local.
